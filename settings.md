## How to open Settings

| Entry | Opens |
|-------|--------|
| **Page overview → Page settings** | Settings for that page (Notifications, App List, …). **Back** returns to page overview. |
| **Page overview → wallpaper control** | **Appearance** (wallpapers, blur/dim, icon packs). Back returns to overview when opened from there. |
| **Full Settings panel** | Root categories (Appearance, Gestures, Wellbeing, Security, Search, Pages, About, …) from the in-app settings UI |

> The **gear on the search bar** opens **page overview**, not Settings.

## Appearance

Compact layout with collapsible live preview:

- **Wallpapers**  
  - System wallpaper, Pitch Black, Bing Daily (Pro), built-in catalog  
  - **Dynamic (day / night)** — import GNOME timed packs; preinstalled **Horizon** pack  
  - **Local images** — system **photo picker**; library of up to **7** photos (tap to use, long-press to remove). Works with wallpaper sync on or off  
  - **Per-page** wallpaper when sync is off in Page overview  
- Theme color swatches  
- Clock style chips, font chips  
- Font scale + icon size dual sliders  
- Hide status bar, network speed, clock background, invert text  
- Sidebar dots / idle fade / letter size; contacts rail can sync or stay independent  
- 24-hour, Fahrenheit

## Security & Privacy

- Uninstallation protection (device admin)  
- Global app locker (accessibility)  
- Finish-setup / limited-protection cards when incomplete  

If accessibility is turned off while locking is expected, Security shows a **Protection is limited** card with a re-enable path.

## Notifications (page settings)

When opened from overview for the Notifications page:

- Colored notification cards  
- **Lock pinned notifications** — pinned apps cannot be swiped away or cleared with Clear All  

## Unified Search

- Web search engine  
- Previous search results / clear history  
- Verification codes (OTP from notifications)  
- Result limit per category  
- **File search** — **Add folder** via the system folder picker (SAF). No broad photo/video library permission. Index rebuild and indexed locations list
