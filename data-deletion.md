**App:** Dextera Launcher (`com.dexteralabs.launcher`)  
**Developer (Play Store):** Dextera Labs  
**Last updated:** July 19, 2026

This page explains how to request deletion of data related to **Dextera Launcher**, published by **Dextera Labs** on Google Play.

---

## Important: most data is only on your device

Dextera is a **device-local home launcher**. Dextera Labs does **not** run a Dextera user account system and does **not** operate a cloud backup of your home layout, contacts, locked apps, or notification contents.

Almost all app data lives **only on your phone** until you remove it.

---

## How to delete your data

### Step 1 — Remove on-device Dextera data (primary method)

Do **one** of the following:

**Option A — Uninstall Dextera Launcher**

1. If you enabled **Uninstallation protection (device admin)**, open Android **Settings → Security → Device admin apps** (wording varies by device), find **Dextera**, and **deactivate** it first.
2. Uninstall **Dextera Launcher** from system settings or the Play Store.

Uninstalling removes Dextera’s app storage, including settings, wallpapers stored by the app, file-search folder grants held by the app, local indexes, and in-app security lists.

**Option B — Clear app storage (keep the app installed)**

1. Android **Settings → Apps → Dextera Launcher → Storage**.
2. Tap **Clear storage** / **Clear data** (and **Clear cache** if shown).

This resets Dextera to a fresh install state on that device.

### Step 2 — Revoke optional permissions (if you keep the app)

In Android **Settings → Apps → Dextera Launcher → Permissions**, deny any access you no longer want (contacts, location, notifications, usage access, etc.).

### Step 3 — Email a deletion request (optional)

If you want a written request on record, or believe some data may have been processed off-device through a feature you used, email:

**Email:** [mike.t.blount@gmail.com](mailto:mike.t.blount@gmail.com)

**Subject line (suggested):** `Dextera Launcher — data deletion request`

**Please include:**

- That your request is about **Dextera Launcher** by **Dextera Labs**
- The Google account email used for Play (optional)
- What you already did (uninstalled / cleared data)
- Any specific concern (e.g. weather or search usage)

We aim to respond within **30 days**.

## What is deleted vs what may be kept

### Deleted when you uninstall or clear app data (on your device)

| Data type | What happens |
|-----------|----------------|
| Launcher settings & preferences | Removed with app data |
| Home layout / page configuration | Removed with app data |
| Local wallpapers stored by Dextera | Removed with app data |
| File-search folder grants & local file index | Removed with app data |
| Locked / hidden / pinned app lists | Removed with app data |
| Notification-related in-app state | Removed with app data |
| Pro unlock state stored in the app | Removed with app data |

### Not controlled by Dextera (you manage separately)

| Data | Notes |
|------|--------|
| **Other apps** you opened from Dextera | Their own data |
| **Android system** accounts, SMS, call logs | Dextera does not own system databases |
| **Weather / Bing / web search providers** | Brief request data may be processed under **their** privacy policies when you use those features. Contact those providers for their processes if needed. |
| **Google Play** purchase/install records | Managed by Google under Play’s policies |

### What Dextera Labs may retain after an email request

| Item | Retention |
|------|-----------|
| Email you send to request deletion | Up to **90 days** for support/compliance, then deleted or anonymized unless law requires longer |
| No Dextera cloud profile | We do **not** keep a server-side copy of your contacts, notification contents, or home layout |

Because we do not operate a Dextera cloud account of user home data, **clearing app data or uninstalling is the complete deletion of Dextera-held app data on the device.**

## Summary

1. **Deactivate device admin** if uninstall protection is on.  
2. **Uninstall** Dextera Launcher **or** **Clear storage**.  
3. Optionally email **mike.t.blount@gmail.com** with subject `Dextera Launcher — data deletion request`.
